## Как запустить локально

Открой `index.html` в браузере.

## Как подключить круглосуточную синхронизацию

1. Открой Google Таблицу.
2. Расширения → Apps Script.
3. Вставь код из `gas-backend.gs`.
4. Нажми Deploy / Развернуть → New deployment / Новое развертывание.
5. Тип: Web app.
6. Execute as: User accessing the web app.
7. Who has access: Anyone with Google account.
8. Скопируй Web App URL.
9. В приложении нажми ⚙ и вставь URL в поле `Apps Script Web App URL`.
10. Нажми `Сохранить` и `Проверить доступ`.

## Важно по доступу

Backend проверяет, есть ли у пользователя доступ редактора к Google Таблице. Если доступа редактора нет, приложение вернёт `Доступ запрещён`.

## Синхронизация кураторов

Приложение каждые 30 секунд перечитывает список кураторов из Google Таблицы. Если в таблице:

- добавили куратора — он появится в приложении;
- удалили куратора — он исчезнет из приложения;
- перенесли куратора между фракциями — он перенесётся и в приложении.

## Публикация на GitHub Pages

1. Создай репозиторий.
2. Загрузи все файлы из архива в корень репозитория.
3. Settings → Pages.
4. Source: Deploy from branch.
5. Branch: main / root.
6. Сохрани.

После этого приложение будет доступно по ссылке GitHub Pages.
